# **Mochi Pets – Animal Shelter Website**

A web‑based adoption platform built for the fictional Mochi Pets shelter in Dubai. This project simulates a complete pet‑adoption workflow, including browsing pets, viewing details, submitting adoption applications, managing shelters, and handling customer interactions.  
It was created as part of a database and web‑development project.

---

## **📌 Features**

### **🐾 Pet Management**
- Separate pages for **cats** and **dogs**
- Individual pet detail pages (breed, age, training, vaccinations, etc.)
- Search functionality for finding pets by name or breed

### **🏠 Shelter Information**
- Shelter listings with organization details  
- Each pet is linked to a specific shelter

### **📝 Adoption System**
- Adoption form for customers  
- “My Applications” page to track submitted adoption requests

### **👤 User Accounts**
- Signup and login system  
- Session‑based login/logout  
- Customer information stored securely

### **👥 Admin & Staff Pages**
- Admin dashboard for managing pets and shelters  
- Staff page for viewing staff roles and responsibilities

### **📄 Additional Pages**
- Home page  
- About page  
- Contact page  
- Privacy policy

## **🎯 Purpose of the Project**
This website was built to demonstrate:
- Database modeling (entities: pets, shelters, customers, staff, adoptions, training, vaccinations)  
- Front‑end design and layout  
- PHP‑based form handling and user authentication  
- Realistic adoption workflow simulation

2. Place the project folder inside the `htdocs` directory  
3. Visit `http://localhost/your-folder-name` in your browser
